## Overview

This STATE.md file serves as the single source of truth for project state.
It is designed to be human-readable and machine-parseable.

### How to Use

- **View state**: Run `swarm state show`
- **Add item**: Run `swarm state add "Task description"`
- **Update item**: Run `swarm state update <id> --status completed`
- **Sync with Hive**: Run `swarm state sync`

### Status Meanings

- `open` - Item is ready to be worked on
- `in_progress` - Item is currently being worked on
- `blocked` - Item is blocked by dependencies or issues
- `completed` - Item is finished
- `cancelled` - Item was cancelled
