## Key Interfaces from Research

### ExpertOutput (dual format pattern)
```typescript
interface ExpertOutput {
  json: {
    findings: Finding[];
    summary: { totalIssues: number; criticalCount: number; filesScanned: number };
    metadata: { durationMs: number; expertVersion: string };
  };
  markdown: string;
}
```

### Security Task Input
```typescript
interface SecurityTaskInput {
  targetPath: string;
  scanTypes: ('secrets' | 'dependencies' | 'patterns')[];
  severityThreshold: 'low' | 'medium' | 'high' | 'critical';
  outputFormat: 'json' | 'markdown' | 'both';
}
```

### Research Libraries to Use
- typhonjs-escomplex: Complexity metrics (cyclomatic, Halstead)
- ts-morph: TypeScript AST manipulation for documentation
- @npmcli/arborist: Dependency vulnerability scanning
