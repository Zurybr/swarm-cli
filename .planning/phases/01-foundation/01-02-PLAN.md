   - complete(params: CompletionParams): Promise<CompletionResult>

2. Type definitions:
   - CompletionParams: { prompt: string; temperature?: number; maxTokens?: number }
   - CompletionResult: { content: string; usage: { prompt: number; completion: number } }

   - ChatResponse: { message: Message; usage: { prompt: number; completion: number } }
   - LLMCall: { params: CompletionParams; timestamp: Date } (for tracking calls)

3. MockLLMClient class implementing LLMClient:
   - Private fixtures: Map<string, CompletionResult>
   - Private calls: LLMCall[] (tracks all calls for verification)
   - Public fallbackMode: 'error' | 'empty' = 'empty'
   - Method setFixture(key: string, response: CompletionResult): void
   - Method setFixturePattern(pattern: RegExp, response: CompletionResult): void
   - Method getCalls(): LLMCall[]
   - Method clearFixtures(): void
   - Method clearCalls(): void
   - Private generateKey(params: CompletionParams): string (hashes normalized prompt)

Reference Pattern 1 from 01-RESEARCH.md.
